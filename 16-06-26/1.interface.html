Interface do Usúario ou UI

O que é Interface de Usúario

Conceito: É a área que se relaciona à criação e ao desenvolvimento da interface gráfica de um produto, serviço ou sistema. O conceito minimalista, por sua vez, é a redução de elementos excessivos para focar apenas no que é mais importante.

Características: Aborda a forma como o usuário irá interagir ou controlar o produto com foco visual e estético. Utiliza a psicologia das cores (já que as cores influenciam fortemente o consumidor) e técnicas para guiar a atenção do usuário. É importante notar que a equipe de UI projeta o gráfico, mas é a equipe de Desenvolvimento que transforma isso em código.

Exemplo: O comparativo visual entre um site minimalista (simples e agradável, que favorece a usabilidade) e um site visualmente poluído.
